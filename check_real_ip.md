# Check the real IP of your Linux machine, directly from your terminal.
1.
```
curl ipecho.net/plain ; echo
```
2.
```
curl -w '\n' ident.me
```
3.
```
curl ifconfig.me
```
4.
```
ifconfig && ccurl ipecho.net/plain ; echo
```
5.
```
ip addr && curl ipecho.net/plain ; echo
```
